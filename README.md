## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/OrbitaTec/Amazonia-lab

# Access
$ cd Amazonia-lab

# Install dependencies
$ npm install

# Run the project
$ npm run dev

# The server will initialize in the <http://localhost:3000>
```
